# 新能车充电管理平台

> 新能车充电管理平台

> gin + mysql + gorm

```
├─src----------------------------父项目
│  │
│  ├─config-------------------------整个程序的配置文件
│  │
│  ├─controller-------------------------存放程序逻辑
│  │
│  │─database------------------------数据库的链接配置(gorm框架)
│  │
│  │─middleware----------------存放中间件文件
│  │
│  │─models----------------存放结构体文件
│  │
│  │─routers----------------存放路由
│  │
│  │─utils----------------工具文件
│  │
│  │─main.go----------------程序入口
```
